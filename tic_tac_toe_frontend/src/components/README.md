# TicTacToe Component

This folder contains the TicTacToe UI for the Remotion frontend.

Highlights:
- Ocean Professional theme (primary #2563EB, secondary #F59E0B, error #EF4444, bg #f9fafb, surface #ffffff, text #111827).
- Centered responsive 3x3 grid with rounded corners, subtle shadows, and smooth transitions.
- Status shows current turn, winner, or draw.
- Winning line highlight and move prevention after game over.
- New Game button to reset state.

The component is mounted in the Remotion Studio as the `TicTacToe` composition via `src/Root.tsx`.
